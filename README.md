# BookSearchSpring
스프링부트로 구현한 책 검색 백엔드 사이드

## H2 DB Table 정보
- BOOK Table
    ```
    CREATE TABLE BOOK(ID INT AUTO_INCREMENT PRIMARY KEY, TITLE VARCHAR(255), AUTHOR VARCHAR(255), KOR_PRICE INT, USA_PRICE DECIMAL, ISBN VARCHAR(20));
    ```
